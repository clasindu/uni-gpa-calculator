* **Dynamic Calculation:** Instantly calculates GPA as you type.

* **Sri Lankan Standards:** Tailored grading scale for local universities.
* **Modern UI:** Glassmorphism design, smooth animations, and a responsive layout.
* **Total Credits:** Automatically sums up your total credits.
